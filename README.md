# 📌 الرصة | The Pinner

Clipboard manager — Windows / Mac / Linux

## التشغيل
```bash
pip install -r requirements.txt
python main.py
```

## البناء كـ exe (Windows)
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "ThePinner" main.py
```

## البناء كـ app (Mac)
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "ThePinner" main.py
```
